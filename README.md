# Lyngurium

A CSS-in-JS library for Lynx.

- ✅ Type-safe by default
- 🚫 No runtime overhead by default
- 📦 Optimized for scale

Lyngurium is a transplant of [Ammolite](https://github.com/gemvale/ammolite) that built for [Lynx](https://github.com/lynx-family). With the same design philosophy, Lyngurium provides the same features as Ammolite, but with a focus on the Lynx ecosystem.
